# List EnviroTox Dataset Names

Lists the names of the envirotox datasets in ssddata.

## Usage

``` r
envirotox_data_sets()
```

## Value

A character vector of envirotox dataset names.

## Examples

``` r
envirotox_data_sets()
#> [1] "envirotox_acute"    "envirotox_chemical" "envirotox_chronic" 
#> [4] "envirotox_data"    
```

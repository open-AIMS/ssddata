# List EnviroTox Dataset Names (Deprecated)

This function was named `list_datasets()` in the standalone `envirotox`
package. It has been renamed to
[`envirotox_data_sets()`](https://open-aims.github.io/ssddata/dev/reference/envirotox_data_sets.md)
in `ssddata`. Please update your scripts to use
[`envirotox_data_sets()`](https://open-aims.github.io/ssddata/dev/reference/envirotox_data_sets.md)
instead.

## Usage

``` r
list_datasets()
```

## Value

A character vector of envirotox dataset names.

## See also

[`envirotox_data_sets()`](https://open-aims.github.io/ssddata/dev/reference/envirotox_data_sets.md)

## Examples

``` r
if (FALSE) { # \dontrun{
# Deprecated - use envirotox_data_sets() instead
list_datasets()
} # }
```
